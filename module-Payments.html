Response:
Name Type Description
status codes.OK if success
redirect_url String if success; is an absolute url to which u need to redirect user;
status codes.WRONG_ARGS if .payment_method_name is not provided
status codes.WRONG_ARGS if .amount_rur_input is listed into the PaymentMethod.requirements but .amount_rur is not provided
status codes.WRONG_ARGS if .phone_number_input is listed into the PaymentMethod but the .phone_number is not provided or is incorrect
status codes.NOT_FOUND if the PaymentMethod with provided .payment_method_name does not exists
status codes.INCORRECT_STATE if PaymentMethod.active is not true
status codes.payments.INCORRECT_DEPOSIT_AMOUNT if you ignore the PaymentMethod.limits content
status codes.INCORRECT_STATE if .steam_id_existence is listed in PaymentMethod.requirements and user has no .steam_id specified
status codes.INCORRECT_STATE if .steam_trade_url_existence is listed in PaymentMethod.requirements and user has no .steam_trade_url specified
desc String on any business-logic level err; it contains the debug information for dev purposes.
