api/affiliates/withdrawals

is a web interface for module:affiliates/withdrawals

Methods

(route) requestAffiliateWithdrawal()

Handlers Chain:
Order
module:middleware/access~addAuthorizationDataToRequest
module:middleware/access~accessUser
this endpoint
Route:
MethodPath
POST /api/affiliates/withdrawals/request

using this endpoint an affiliate can request a money transfer to his gaming platform balance

Body Parameters:
Name Type Description
usd_amount Number amount to transfer to gaming platform balance
Response:
Name Type Description
status OK if success
affiliate_withdrawal AffiliateWithdrawal if success, see AffiliateWithdrawal
updated_affiliate_balance Number if success
undefined payments.INCORRECT_WITHDRAW_AMOUNT if withdraw amount is too high or too low
undefined payments.INSUFFICIENT_FUNDS if .usd_amount > affiliate balance
status FORBIDDEN if not authorized
status FAIL if internal error

Handlers Chain:
Order
module:middleware/access~addAuthorizationDataToRequest
module:middleware/access~accessAdmin - required permission scopes: AFFILIATES, SUPPORT
this endpoint
Route:
MethodPath
GET /api/affiliates/withdrawals/list

returns list of affiliate withdrawals

Query Parameters:
Name Type Attributes Description
affiliate_id Number optional to filter withdrawals by Affiliate's .affiliate_id
username String optional to filter withdrawals by Affiliate's .username
affiliate_withdrawal_kind AffiliateWithdrawalKinds optional see AffiliateWithdrawalKinds
status AffiliateWithdrawalStatuses optional see AffiliateWithdrawalStatuses
offset Number optional default is 0
limit Number optional default is 15
sorting_property String optional default is 'u.created_at', old affiliates first
sorting_direction String optional 'DESC' or 'ASC', default is 'ASC'
Response:
Name Type Description
status OK if success
affiliate_withdrawals Array.<AffiliateWithdrawal> if success, see AffiliateWithdrawal
status FORBIDDEN if auth cookie doesnt contain the required permissions scope(s)
status FAIL if internal error
