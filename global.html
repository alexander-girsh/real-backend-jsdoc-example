hasAccessAdmin(post_auth, response, scope) → {boolean}

Source:
  • broadcasting-server/middleware/ws-auth.js, line 98

Method implements the http-like authorization. It supports admin_scope like the http version. Unfortunately we do not have middlewares on our simply router so we need to use boolean result of this function call

Parameters:
Name Type Description
post_auth Object
response Object
scope Object

here u should specify the admin_scope required for endpoint use
Returns:

If false, u can break the endpoint work, cause response with status: codes.FORBIDDEN will be send automatically

Usage:

function someEndPoint ({payload, post_auth, response}) {

if (!hasAccessAdmin({post_auth, response, scope: { 'FINANCE_PROCESSING': true } })) {
    return  // response with 'forbidden' status will be sent already. U can end the handling immediately.
}

}
